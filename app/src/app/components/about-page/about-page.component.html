Spec Math

Google Summer 2020 intern project built under the Google Cloud Apigee platform.

What is Spec Math?

Spec Math is a service for performing operations on OpenAPI specifications. OpenAPI is an industry-standard format for machine-readable REST API descriptions.

Specifications play a critical role in the API lifecycle. They can enforce a contract between client and server teams, easily generate documentation and client libraries, reduce the work of creating a proxy for an existing service and much more. The Spec Math project aims to afford even more power to API developers who use the OpenAPI specification.

How does it work?

We use tree traversal algorithms to represent a YAML file and have the ability to compare or edit each of these "nodes".

What can it do?

Spec Math supports a series of operations such as

Merge

Combine a set of spec files into a new spec. Learn more

Overlay

Replace a set of attributes or paths on a spec using a spec fragment. Learn more

Filter

Extract a set of metadata from a spec to produce a new one. Learn more
